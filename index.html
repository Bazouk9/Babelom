Le projet

Nous naissons quelque part. Nous grandissons quelque part. Nous partons, revenons, repartons. Chaque lieu laisse une empreinte — dans nos corps, dans notre façon de parler, dans ce que nous appelons chez nous.

Pourtant, ces traces disparaissent. Les maisons changent de locataires, les quartiers se transforment, les familles se dispersent. Dans cent ans, qui saura que vous avez vécu là ?

Babelom est né de cette question. C'est une carte mondiale où chacun peut épingler les lieux où il a habité — avec ses dates, ses anecdotes, ses photos. Un point sur la carte. Une vie derrière ce point.

Ensemble, nous construisons une mémoire collective des lieux habités — pour les chercheurs, pour les familles, pour les générations qui n'ont pas encore de mémoire à transmettre.

Mode d'emploi

Trois gestes simples
pour une trace durable

01
Cliquez sur la carte

Naviguez jusqu'au lieu exact où vous avez vécu — rue, quartier, village. Cliquez pour y planter votre point.
02
Racontez

Les dates, le type de logement, mais surtout les anecdotes — l'odeur de la cage d'escalier, le bruit du voisin, la vue depuis la fenêtre.

03
Transmettez

Votre souvenir rejoint l'archive. Accessible à vos enfants, à vos petits-enfants, aux chercheurs — pour toujours.
